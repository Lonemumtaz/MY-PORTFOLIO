# MY-PORTFOLIO

Personal Portfolio Website for **Mumtaz Lone**

## Description
A modern, responsive portfolio website showcasing the projects, skills, and experience of Mumtaz Lone — Full Stack Developer & Data Analyst from Srinagar, J&K, India.

## 🚀 Features
- Responsive navigation bar and smooth scrolling
- Hero section with profile image upload (localStorage support)
- About section with education, focus, and languages
- Skills section (Programming, Frontend, Data Analysis, Core Concepts, Tools, Soft Skills)
- Projects showcase:
	- Dictionary App ([GitHub](https://github.com/Lonemumtaz/Dictionary-App))
	- Netflix Data Analysis ([GitHub](https://github.com/Lonemumtaz/NETFLIX_DataAnalysis_Project))
	- BMI Calculator ([GitHub](https://github.com/Lonemumtaz))
	- Ludo Python Game ([GitHub](https://github.com/Lonemumtaz/ludo-python-game))
	- Library Management System ([GitHub](https://github.com/Lonemumtaz))
	- Rock Paper Scissors ([GitHub](https://github.com/Lonemumtaz/RockScissor))
- Hackathon experience section
- Contact section with email, phone, and social links
- Animations and interactive UI (IntersectionObserver, counter, mobile menu)

## 🛠️ Tech Stack
- HTML5, CSS3 (custom styles, responsive design)
- JavaScript (DOM manipulation, localStorage, IntersectionObserver)
- Font Awesome (icons)
